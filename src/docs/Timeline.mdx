# Usage

## Simple Example 

<Playground>
  <Chart 
    width={"500px"}
    height={"300px"}
    chartType="Timeline"
    loader={<div>Loading Chart</div>}
    data={[
      [
        { type: 'string', id: 'President' }, 
        { type: 'date', id: 'Start' },
        { type: 'date', id: 'End' }

    options={{
      showRowNumber:true
    }}
  />
</Playground>
