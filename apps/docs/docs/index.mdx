# React Google Charts

<img src={logoUrl} alt="Logo" className="logo" />

React Google Charts is a lightweight and fully typed React wrapper for [Google Charts](https://developers.google.com/chart/interactive/docs/reference).

It's easy to use, supports over 25 chart types, supports animations and is highly customizable when needed.

## Quickstart

Install this library with your favorite package manager:

```bash
npm install --save react-google-charts
```

Then, import the Chart component

```js
import { Chart } from "react-google-charts";
```

and use it:

```jsx live
function MyChart(props) {
  return (
    <Chart
      // Try different chart types by changing this property with one of: LineChart, BarChart, AreaChart...
      chartType="ScatterChart"
      data={[
        ["Age", "Weight"],
        [4, 16],
        [8, 25],
        [12, 40],
        [16, 55],
        [20, 70],
      ]}
      height="400px"
      options={{
        hAxis: {
          maxValue: 22,
          title: "Age (Years)",
        },
        vAxis: {
          maxValue: 90,
          title: "Weight (kg)",
        },
        title: "Average Weight by Age",
      }}
      legendToggle
    />
  );
}
```

You can also use the material design versions available for [Bar Charts](/examples/bar-chart-material) and [Line Charts](/examples/bar-chart-material)

```jsx live
function App() {
  const data = [
    ["Year", "Sales", "Expenses", "Profit"],
    ["2014", 1000, 400, 200],
    ["2015", 1170, 460, 250],
    ["2016", 660, 1120, 300],
    ["2017", 1030, 540, 350],
  ];
  const options = {
    chart: {
      title: "Company Performance",
      subtitle: "Sales, Expenses, and Profit: 2014-2017",
    },
  };
  return (
    <Chart
      chartType="Bar"
      width="100%"
      height="400px"
      data={data}
      options={options}
    />
  );
}
```
